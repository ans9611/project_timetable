# TimeTable Maker
## 과목명과 과목별 시간표들을 입력하면 모든 조합가능 경우를 보여줌

1. 수업 이름들 입력 후 'next' 클릭
2. 각 수업별로 듣고싶은 시간표 입력후 'SET' 클릭 (클릭할때마다 추가됨)
3. 시간표 탭으로 이동후 'show' 클릭
4. 'options' 에서 시간표들을 비교, 더블클릭으로 option 이름을 바꿀수 있다.
5. 'info' 클릭시 해당 시간표에대한 정보 보여줌.
